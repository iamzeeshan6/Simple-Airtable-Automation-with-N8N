# Simple-Airtable-Automation-with-N8N
n8n automation that monitors a Google Drive folder, summarizes new documents using GPT, and stores the summaries in Airtable for easy reference.

<img width="649" height="158" alt="image" src="https://github.com/user-attachments/assets/9f695dc0-0237-4b8c-ac9e-3948f142af0c" />

This repository contains a simple yet powerful n8n automation that automatically processes new documents added to a specific Google Drive folder. Whenever a new file is created, the workflow retrieves the document content, sends it to GPT to generate a concise summary, and stores the summarized output in Airtable. This eliminates the need to manually open, read, and review every document, saving time and improving knowledge management.
